## NexNav Component

`NexNav `is a component designed to create navigation bars or menus within your React application. It offers the following features:

* **Navigation Menu:** `NexNav` displays a customizable navigation menu with logo, menu items, and optional identity links (e.g., login, sign up).
* **Mobile-Friendly:** The component includes a responsive design, making it suitable for desktop and mobile devices.
* **Customization:** You can customize the appearance and behavior of the navigation menu by adjusting various props and styles.
* **Accessibility:** The component ensures accessibility standards are met, providing a seamless navigation experience for all users.

### Usage

To use `NexNav`, simply import the component and provide the necessary props:

```tsx
import React from 'react';
import NexNav from '@nexcomponent/lib';

function MyComponent() {
  const navItems = [
    { label: 'Home', onClick: () => console.log('Home clicked') },
    { 
      label: 'Products', 
      subItems: [
        { label: 'Overview', onClick: () => console.log('Products Overview clicked') },
        { label: 'Features', onClick: () => console.log('Features clicked') }
      ]
    },
    { label: 'About', onClick: () => console.log('About clicked') },
  ];

  return (
    <NexNav
      logoSrc="/path/to/logo.png"
      displayName="Company Name"
      navItems={navItems}
      homeButtonHandler={() => console.log('Home clicked')}
      isAuthenticated={true}
      user={{ name: 'John Doe' }}
      onLogout={() => console.log('Logout clicked')}
      languageOptions={[
        { code: 'en', label: 'English' },
        { code: 'es', label: 'Español' }
      ]}
    />
  );
}

export default MyComponent;
```

### Props

* **logoSrc (optional):** URL of the logo image to be displayed in the navigation bar.
* **displayName (required):** Alternate text for the logo image, or the company name if no logo is provided.
* **homeButtonHandler (required):** Function called when the home/logo button is clicked.
* **navItems (required):** Array of menu items to be displayed in the navigation menu. Each item should have a `label` and an optional `onClick` function (not used when `subItems` are present).
* **isAuthenticated (required):** Boolean indicating if the user is authenticated.
* **user (optional):** User object with name, avatarUrl, and role properties.
* **onLogin (optional):** Function called when login button is clicked.
* **onLogout (optional):** Function called when logout button is clicked.
* **onProfile (optional):** Function called when profile button is clicked.
* **languageOptions (required):** Array of language options for the language switcher.

### NavItem Behavior

* **Nav items with sub-items:** Hover to open dropdown, click does nothing
* **Nav items without sub-items:** Click to navigate directly
* **Sub-items in dropdown:** Click to navigate to specific pages/sections

### Customization

`NexNav `can be customized by adjusting the styles defined in the accompanying CSS file (`NexNav.scss`). You can modify the appearance, layout, and behavior of the navigation menu to match your application's design requirements.
