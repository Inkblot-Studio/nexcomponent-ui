# NexFooter

A modern, scalable footer component with glassmorphic styling that perfectly complements the NexNav design system. Built with 2025 design trends, clean animations, and enterprise-grade features.

## Features

- **2025 Design Trends**: Modern glassmorphic styling with subtle gradients and clean typography
- **Contact Form Integration**: Built-in contact form for lead generation and customer support
- **Newsletter Signup**: Email subscription with validation and loading states
- **Social Media Integration**: Properly sized social icons with platform-specific styling
- **Multiple Variants**: Default, contact, and black glass layouts
- **Responsive Design**: Mobile-first approach with adaptive grid layouts
- **Accessibility**: WCAG compliant with reduced motion support
- **Theme Support**: Light, dark, and auto theme variants

## Usage

```tsx
import { NexFooter } from '@nexcomponent/ui';

function App() {
  return (
    <NexFooter
      displayName="Your Company"
      tagline="Building amazing products"
      variant="default"
      newsletter={{
        enabled: true,
        placeholder: "Stay updated",
        onSubmit: async (email) => {
          // Handle newsletter subscription
        }
      }}
      contact={{
        enabled: true,
        title: "Get in Touch",
        description: "We'd love to hear from you",
        onSubmit: async (data) => {
          // Handle contact form submission
        }
      }}
      socials={[
        { type: 'github', url: 'https://github.com/yourcompany' },
        { type: 'twitter', url: 'https://twitter.com/yourcompany' }
      ]}
    />
  );
}
```

## Customization

### Styling
The footer uses CSS custom properties for easy theming:

```scss
.nex-footer {
  --nex-footer-background: rgba(255, 255, 255, 0.06);
  --nex-footer-border: rgba(255, 255, 255, 0.08);
  --nex-footer-shadow: 0 4px 20px rgba(0, 0, 0, 0.1);
}
```

### Social Icons
Add custom social platforms by extending the icon mapping:

```tsx
const customSocialIcons = {
  ...socialIcons,
  custom: () => <YourCustomIcon />
};
```

### Contact Form
Customize the contact form behavior and styling:

```tsx
contact={{
  enabled: true,
  title: "Custom Title",
  description: "Custom description",
  placeholder: "Custom placeholder",
  buttonText: "Custom Button",
  onSubmit: async (data) => {
    // Custom submission logic
  }
}}
```

## Migration Guide

### From v1.x to v2.x

- `variant` prop now supports `'contact'` option
- `contact` prop added for contact form functionality
- Social icons now have fixed sizing (16px default)
- Improved hover animations matching navbar patterns
- Enhanced glassmorphic styling with 2025 design trends

```tsx
// Old
<NexFooter variant="compact" />

// New
<NexFooter 
  variant="contact"
  contact={{
    enabled: true,
    onSubmit: handleContact
  }}
/>
```
