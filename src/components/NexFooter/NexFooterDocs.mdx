# NexFooter

Enterprise-grade footer component with glassmorphic styling that complements the NexNav design system. Features newsletter signup, developer tools, social media integration, and responsive design.

## Features

- **Glassmorphic Design**: Matches NexNav styling with backdrop filters and subtle elevation
- **Newsletter Integration**: Email capture with validation and submission handling
- **Developer Tools**: NPM package commands with copy-to-clipboard functionality
- **Social Media**: Configurable social links with platform-specific styling
- **Responsive Layout**: Adaptive grid system for all screen sizes
- **Accessibility**: Full keyboard navigation and screen reader support
- **Framer Motion**: Smooth animations with reduced motion support

## Usage

```tsx
import { NexFooter } from '@nexcomponent/ui';

function App() {
  const handleNewsletterSubmit = async (email: string) => {
    // Handle newsletter subscription
    await subscribeToNewsletter(email);
  };

  return (
    <NexFooter
      displayName="NexComponent"
      tagline="Professional React components for modern applications"
      sections={[
        {
          title: 'Documentation',
          links: [
            { label: 'Getting Started', url: '/docs/getting-started' },
            { label: 'Components', url: '/docs/components' }
          ]
        }
      ]}
      newsletter={{
        enabled: true,
        placeholder: 'Stay updated',
        onSubmit: handleNewsletterSubmit
      }}
      socials={[
        { type: 'github', url: 'https://github.com/nexcomponent' },
        { type: 'twitter', url: 'https://twitter.com/nexcomponent' }
      ]}
      developerTools={{
        npmPackage: '@nexcomponent/ui',
        githubUrl: 'https://github.com/nexcomponent/ui'
      }}
    />
  );
}
```

## Design System Integration

The NexFooter component follows the same design principles as NexNav:

- **Glassmorphic Styling**: Uses backdrop filters and subtle gradients
- **Token-Based Design**: All styling uses design system tokens
- **Consistent Spacing**: Follows the 4px base unit system
- **Typography**: Uses the same font families and weights
- **Color System**: Integrates with light/dark theme tokens
- **Animation**: Uses Framer Motion with centralized configuration

## Accessibility

- **Semantic HTML**: Uses proper footer and navigation elements
- **Keyboard Navigation**: Full keyboard support for all interactive elements
- **Screen Readers**: Proper ARIA labels and semantic structure
- **Focus Management**: Clear focus indicators and logical tab order
- **Reduced Motion**: Respects user motion preferences

## Performance

- **Optimized Animations**: Uses `will-change` and hardware acceleration
- **Lazy Loading**: Non-critical content loads efficiently
- **Minimal Bundle**: Lightweight component with tree-shaking support
- **Efficient Rendering**: Optimized re-renders with React best practices

## Browser Support

- **Modern Browsers**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Backdrop Filter**: Graceful fallback for older browsers
- **CSS Grid**: Responsive layout with fallback support
- **ES6+ Features**: Modern JavaScript with Babel compilation
