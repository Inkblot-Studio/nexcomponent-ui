NexCard Component

`NexCard` is a versatile component used to display cards with various content within your React application. It offers the following features:

* **Content Display:** `NexCard` supports displaying images, titles, text content, actions, and footer sections.
* **Customization:** You can customize the appearance of the card, including the presence of images, titles, and footer sections.
* **Flexibility:** `NexCard` can be easily integrated into any React application and styled to match your design system.
* **Accessibility:** The component ensures accessibility standards are met, providing a seamless experience for all users.

### Usage

##### Basic Usage

To use `NexCard`, simply import the component and provide the necessary props:

```tsx
import React from 'react';
import NexCard from '@nexcomponent/lib';

function MyComponent() {
  return (
    <NexCard
      title="Card Title"
      image="path/to/image.jpg"
      content="Lorem ipsum dolor sit amet, consectetur adipiscing elit."
      actions={<button onClick={() => console.log('Action clicked')}>Action</button>}
      footer={<div>Card Footer</div>}
    />
  );
}

export default MyComponent;
```

### Props

* **title (optional):** Title of the card.
* **image (optional):** Image URL to be displayed on the card.
* **content (optional):** Text content to be displayed on the card.
* **actions (optional):** Actions to be displayed on the card (e.g., buttons, links).
* **footer (optional):** Footer content to be displayed below the card.
