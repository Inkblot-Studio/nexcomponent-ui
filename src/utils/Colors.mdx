# Theming Guide for NexComponent

## Overview

NexComponent allows for extensive customization to ensure that your UI components align perfectly with your brand identity and design requirements. By leveraging CSS variables, you can easily adjust the colors and styles of the components throughout your application.

## Customizing Colors

NexComponent uses a set of predefined color variables to maintain consistency. You can customize these colors at the root level of your project to reflect your desired palette.

### Default Color Variables

@import url('https://fonts.googleapis.com/css2?family=Inter:wght@100..900&display=swap');

  /* Primary colors */
  --nex-primary-color: #3D4DE1;
  --nex-primary-color-rgb: 61, 77, 225;

### Customization Steps

1. **Root Level Customization**: Modify the color variables within the `:root` selector to match your brand's color scheme:

```css
:root {
  /* Primary colors */
  --nex-primary-color: #1f9be1;
  --nex-primary-contrast-color: #f3f3f3;

  /* Secondary colors */
  --nex-secondary-color: #ef910a;
  --nex-secondary-contrast-color: #f3f3f3;

  /* Tertiary colors */
  --nex-tertiary-color: #88ffca;
  --nex-tertiary-contrast-color: #f3f3f3;

  /* Quaternary colors */
  --nex-quaternary-color: #4addef;
  --nex-quaternary-contrast-color: #f3f3f3;

  /* Font color */
  --nex-font-color: #1e1d2a;

  /* Background color */
  --nex-background-color: #f3f3f3;

  /* Complimentary color */
  --nex-complimentary-color: #bababa;

  /* Bootstrap */
  --nex-success: #5cb85c;
  --nex-info: #5bc0de;
  --nex-warning: #f0ad4e;
  --nex-danger: #d9534f;
}
```

1. **Changing Colors**: Adjust the following variables to tailor the color scheme to your project's needs:
   * **Primary Colors**: Update `--nex-primary-color` and `--nex-primary-contrast-color` for primary elements and their contrast.
   * **Secondary Colors**: Modify `--nex-secondary-color` and `--nex-secondary-contrast-color` for secondary elements.
   * **Tertiary and Quaternary Colors**: Customize `--nex-tertiary-color`, `--nex-tertiary-contrast-color`, `--nex-quaternary-color`, and `--nex-quaternary-contrast-color` as needed.
   * **Font and Background Colors**: Adjust `--nex-font-color` and `--nex-background-color` to match your text and background preferences.
   * **Complimentary Color**: Change `--nex-complimentary-color` for accents and highlights.
   * **Bootstrap Colors**: Optionally, modify `--nex-success`, `--nex-info`, `--nex-warning`, and `--nex-danger` to seamlessly integrate with Bootstrap components.

### Applying Your Custom Theme

After updating the variables, ensure your styles are correctly applied by importing your custom CSS file into your project:

```css
import './path/to/your/custom.css';
```

By following these steps, you can effortlessly tailor the color scheme of your NexComponent library to match your brand and project requirements.

For further guidance or assistance, refer to the detailed documentation of NexComponent or consult with your development team.
